### 데이터 팀 사내 스터디를 위한 레포지토리
- 매주 월요일 7시에 시작합니다
- 시작과 동시에 책을 열심히 읽습니다
- 모두가 다 읽었으면 한 명씩 인상적인 부분을 말하거나 질문을 합니다
